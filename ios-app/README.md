# WhereUAt iOS App

## Project Overview
Mobile application for the WhereUAt platform, built with React Native.

## Prerequisites
- Node.js 14+
- React Native CLI
- Xcode (for iOS development)
- CocoaPods

## Setup
1. Clone the repository
2. Run `npm install`
3. For iOS: `cd ios && pod install`
4. Run `npm run ios`
